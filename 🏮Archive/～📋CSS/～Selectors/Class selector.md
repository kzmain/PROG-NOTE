# Description:
To select all nodes who has specific class nodes by "."

# Structure
```
.CLASSVALUE{
	
}
```

# Example
```
.intro {
    color: red;
    font-weight: bold;
}

<div id="top">

<h1>Chocolate curry</h1>

<p class="intro">This is my recipe for making curry purely with chocolate</p>

<p class="intro">Mmm mm mmmmm</p>

</div>
```
