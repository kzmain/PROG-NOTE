# Usage
1. If a string +r+contains Another String+r+
2. If an element is in a list
3. If a +r+dictionary contains a key+r+

# Structure
```
A IN B
```

# Example
```
>>> str = "Messi is the best soccer player"
>>> "soccer" in str
True
>>> "football" in str
False
```
