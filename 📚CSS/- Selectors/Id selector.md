# Description:
To select all nodes who has specific id by "#"

# Structure
```
#ID_VALUE{
	
}
```

# Example
```
#top {
    background-color: #ccc;
    padding: 20px
}

<div id="top">

<h1>Chocolate curry</h1>

<p class="intro">This is my recipe for making curry purely with chocolate</p>

<p class="intro">Mmm mm mmmmm</p>

</div>
```
