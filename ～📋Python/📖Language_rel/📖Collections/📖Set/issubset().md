# Description
The `issubset()` method returns `True` if all items in the set exists in the specified set, otherwise it retuns `False`.

# Syntax
```
COMPARE_SET.issubset(TARGET_SET)
```

# Example 
```
x = {"a", "b", "c"}
y = {"f", "e", "d", "c", "b"}

z = x.issubset(y) 

print(z)
```
