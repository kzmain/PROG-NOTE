# Description
Remove a class atribute from class list

# Syntax
```
DOM_ELEMENT.classList.remove(CLASS_VALUE)
```

# Example
```
function myFunction() {
  var element = document.getElementById("myDIV");
  element.classList.remove("mystyle");
}
```
