# Description:
To remove an element's +r+first occurrence in a list+r+

# Type:
1. Reverse sort, sort a list in descending order
  `LIST_NAME.remove(ELEMENT_TO_REMOVE)`

# Example:
Input
```
a = ["a", "a", "n"]
a.remove("a")
print(a)
```

Output
```
['a', 'n']
```
