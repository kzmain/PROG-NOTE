# Description:
1. Include javascripts code in specific `.html` file
2. The src attribute specifies the URL of an external script file.

# Usage:
  1. Infile scripts
  ```
  <script type="text/javascript">
  	STATEMENT 1
  	STATEMENT 2
  	STATEMENT 3
  </script>>
  ```
  2. Include external script file
  `<script src="myscripts.js"></script>`
